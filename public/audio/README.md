# Audio Files for Valentine's Landing Page

## Required Audio Files

Please add the following audio files to this directory:

### 1. Background Music
- **Filename**: `background-music.mp3`
- **Type**: Romantic instrumental or her favorite love song
- **Duration**: 2-5 minutes (will loop)
- **Recommendations**:
  - "A Thousand Years" (instrumental)
  - "Can't Help Falling in Love" (instrumental)
  - Any soft piano/acoustic instrumental
- **Where to get**: YouTube (use a converter), Spotify downloads, or royalty-free music sites

### 2. Heartbeat Sound
- **Filename**: `heartbeat.mp3`
- **Type**: Subtle heartbeat sound effect
- **Duration**: 1-2 seconds
- **Where to get**: [Freesound.org](https://freesound.org/search/?q=heartbeat), [Zapsplat](https://www.zapsplat.com/)

### 3. Celebration Sound
- **Filename**: `celebration.mp3`
- **Type**: Cheerful celebration/success sound
- **Duration**: 2-4 seconds
- **Recommendations**: Chimes, bells, or happy fanfare
- **Where to get**: [Freesound.org](https://freesound.org/search/?q=celebration), [Zapsplat](https://www.zapsplat.com/)

## Supported Formats
- MP3 (recommended - best browser support)
- WAV (higher quality but larger file size)
- OGG (good alternative)

## Tips
- Keep file sizes reasonable (background music < 5MB, effects < 500KB)
- Test volume levels - effects should be quieter than background music
- Ensure audio is high quality but compressed for web
